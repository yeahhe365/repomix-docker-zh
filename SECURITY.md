# Security Policy

## Reporting a Vulnerability

To securely report a vulnerability in this fork, please [open an advisory on GitHub](https://github.com/yeahhe365/repomix-local-webui/security/advisories/new).

If the issue affects the upstream Repomix project, please also report it to [yamadashy/repomix](https://github.com/yamadashy/repomix/security/advisories/new).
